
# Sannsynlighetsnettside for 9. trinn

Dette er en lettvekts prototype i ren HTML, CSS og JavaScript.

## Innhold
- `index.html` : forside med læringsstier
- `formler.html` : formelveiviser og små formelhjelpere
- `test.html` : testgenerator med øvingsmodus og prøvemodus
- `programmering.html` : Python-editor med eksempelkode for mynter og terninger
- `tema/` : undersider for hvert tema

## Tekniske valg
- Ingen innlogging
- Ingen backend
- Ingen tunge biblioteker på vanlige sider
- CodeMirror + Skulpt lastes bare på programmeringssiden
- Kalkulatoren er hjemmelaget og støtter `!`, `nCr`, `nPr` og `^`

## Tips
- Åpne `index.html` i nettleser for å starte
- Programmeringssiden trenger nett for å hente CDN-filene til Skulpt og CodeMirror
- Resten av sidene er vanlige statiske filer
